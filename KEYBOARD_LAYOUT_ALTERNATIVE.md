# Huawei Glide Magyar Billentyűzet - Alternatív Kiosztás

Ez a módosított billentyűzetkiosztás a Huawei Glide fizikai billentyűzethez, magyar QWERTZ alapon.

## Fő Változások az Eredeti Kiosztáshoz Képest

1. **Escape → 0**: Az ESC billentyű most "0" karaktert ír
2. **0 → ö/Ö**: A 0 billentyű most "ö" vagy "Ö" karaktert ír

## Billentyűzet Váltás

- **Shift + Space**: Váltás az eredeti kiosztásra

## Magyar Karakterek Gyorselérése (Alternatív Kiosztás)

### Három lehetőség az "ö" beírására:
1. **0 billentyű** → ö (alap)
2. **Shift+0** → Ö (nagybetű)
3. **Alt+O** vagy **Shift+Alt+O** → ö/Ö

### Egyéb magyar karakterek:
- **ő**: [ billentyű
- **ú**: ] billentyű
- **ű**: \ billentyű
- **ü**: - (Minus) billentyű
- **ó**: = (Equals) billentyű
- **é**: ; billentyű
- **á**: ' billentyű

### A "0" szám beírása:
- **ESC billentyű** → 0

## Használati Tippek

Ez a kiosztás akkor hasznos, ha:
- Gyakran használod az **ö** karaktert
- A **0 billentyű** könnyebben elérhető, mint az ESC
- Ritkán használod az ESC funkciót (szövegírásnál)

## Speciális Karakterek

- **€ (Euro)**: Alt+U
- **@ (kukac)**: Alt+V
- **# (hashtag)**: Alt+X
- **& (és jel)**: Alt+C
- **$ (dollár)**: Alt+;
- **\ (backslash)**: Alt+Q
- **| (pipe)**: Alt+W
- **[ ] (szögletes zárójelek)**: Alt+F, Alt+G
- **{ } (kapcsos zárójelek)**: Alt+B, Alt+N

## QWERTZ vs QWERTY Különbség

Ez a kiosztás **QWERTZ** alapú (német/magyar szabvány):
- A fizikai **Y** billentyű → **Z** karaktert ír
- A fizikai **Z** billentyű → **Y** karaktert ír

## Ékezetes Karakterek (Diakritikus jelek)

Shift+Alt kombinációkkal érhetők el:
- **~ (tilde)**: Shift+Alt+1
- **ˇ (háček)**: Shift+Alt+2
- **^ (circumflex)**: Shift+Alt+3
- **˘ (brevé)**: Shift+Alt+4
- **° (fok jel)**: Shift+Alt+5
- **˛ (ogonek)**: Shift+Alt+6
- **` (grave)**: Shift+Alt+7
- **˙ (pont felül)**: Shift+Alt+8
- **´ (éles ékezet)**: Shift+Alt+9
- **˝ (dupla éles ékezet)**: Shift+Alt+0 → **Figyelem:** Ez most "ö" karaktert ír Shift+Alt+0 helyett!
- **¨ (trema)**: Shift+Alt+-
- **¸ (cedilla)**: Shift+Alt+=
